# UPPERCASE-BOOT
UPPERCASE-BOOT는 UPPERCASE의 모든 모듈을 불러들이고, 프로젝트를 실행하기 위해 필요한 모듈입니다. 구동을 위해 [UPPERCASE-MODEL](UPPERCASE-MODEL.md)이 필요합니다.
